Image Searcher

This API can search with Google's Open Search API for images and return the url of the image, the name of it, its thumbnail, and the context of the image.

To call for a search, make a call in this format: https://image-searching.glitch.me/api/imagesearch/x where x is your search term.

To look at most recent searches and when they occured, make this call: https://image-searching.glitch.me/api/latest/imagesearch/
